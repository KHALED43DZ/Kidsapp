# 🎬 AI Kids Cartoon Studio – Mobile PWA

## ملفات المشروع
```
index.html      ← التطبيق الرئيسي (كل شيء في ملف واحد)
manifest.json   ← إعدادات PWA
sw.js           ← Service Worker للعمل أوفلاين
README.md       ← هذا الملف
```

## 🚀 النشر على GitHub Pages
1. أنشئ مستودع جديد على GitHub
2. ارفع الملفات الثلاثة: `index.html`, `manifest.json`, `sw.js`
3. فعّل GitHub Pages من الإعدادات (Settings → Pages → main branch)
4. التطبيق جاهز على: `https://USERNAME.github.io/REPO/`

## 🔑 مفاتيح API المطلوبة

| API | الاستخدام | التكلفة |
|-----|-----------|---------|
| **Anthropic Claude** | توليد السيناريو والشخصيات | مدفوع (مطلوب) |
| **Hugging Face** | توليد الصور (FLUX.1) | مجاني |
| **OpenAI (اختياري)** | صور DALL-E 3 (احتياطي) | مدفوع |

### الحصول على المفاتيح:
- Anthropic: https://console.anthropic.com/
- Hugging Face: https://huggingface.co/settings/tokens
- OpenAI: https://platform.openai.com/api-keys

## ⚙️ الميزات

### الاستوديو 🎨
- إدخال فكرة المسلسل
- رفع صور مرجعية للشخصيات
- اختيار نوع المحتوى (تربوي/تشويقي/ترفيهي/حماسي/مغامرات/كوميدي)
- اختيار الفئة العمرية (2-14 سنة)
- اختيار أسلوب الرسم (2D/3D/أنيمي/ألوان مائية)
- أبعاد الفيديو (9:16 للـ Shorts / 16:9 لـ YouTube)
- التحكم في عدد الحلقات (1-12 حلقة)
- التحكم في مدة كل حلقة (1-15 دقيقة)
- ضبط عدد المشاهد (تلقائي أو يدوي)

### النتائج 📦
- عرض الحلقات مع المشاهد
- عرض صور المشاهد (مع توليد فردي لكل مشهد)
- عرض الحوارات المشكولة
- نص صوتي مع TTS عبر المتصفح
- تصدير: ZIP لكل حلقة، السيناريوهات، النصوص الصوتية

### الشخصيات 👤
- Character Bible لكل شخصية
- وصف تفصيلي + بروميت توليد الصور
- بيانات ثابتة لضمان الاتساق

### الإعدادات ⚙️
- إدارة مفاتيح API (مخزنة في localStorage)
- اختبار الاتصال
- وضع التوفير (تقليل طلبات API)
- التحكم في التشكيل التلقائي

## 📱 التثبيت كـ PWA
- Android: "إضافة إلى الشاشة الرئيسية" من Chrome
- iOS: "Add to Home Screen" من Safari

## 🔄 Pipeline العمل
```
فكرة المستخدم
     ↓
[Claude API] → سيناريو كامل (JSON)
     ↓
[Claude API] → Character Bible
     ↓
[HF / OpenAI API] → صور المشاهد
     ↓
تنظيم الحزمة + تصدير
```

## ⚠️ ملاحظات
- يعمل بالكامل من المتصفح بدون تثبيت
- جميع البيانات محفوظة في localStorage
- يعمل أوفلاين بعد أول تحميل (بدون توليد جديد)
- متوافق مع Chrome, Firefox, Safari, Edge
